# Functionality Docs

Plain-language descriptions of every feature in FundsFlee, written for non-developer users.

---

| File | What it covers |
|------|----------------|
| [analysis.md](analysis.md) | Generates a breakdown of your spending by category with AI insights and saving tips |
| [auth-session.md](auth-session.md) | Signs you in with Google and keeps your session alive automatically |
| [bank-statement-import.md](bank-statement-import.md) | Uploads a PDF bank statement and extracts all your debit transactions |
| [categories.md](categories.md) | View built-in spending categories and create your own custom ones |
| [compare-prices.md](compare-prices.md) | Compares two to four merchants side by side using AI scoring |
| [dashboard.md](dashboard.md) | Home screen with spending chart, period filter, recent transactions, and recurring reminders |
| [data-export.md](data-export.md) | Export transactions as CSV, open the raw sheet, or clear cached data |
| [duplicate-detection.md](duplicate-detection.md) | Finds transactions that were recorded more than once from different sources |
| [email-import.md](email-import.md) | Reads bank and payment emails from Gmail and adds the transactions automatically |
| [item-suggestions.md](item-suggestions.md) | Suggests clean-ups for inconsistent item names and fills in missing details |
| [manual-transaction.md](manual-transaction.md) | Add an expense by hand using a number pad and simple form fields |
| [merge-duplicates.md](merge-duplicates.md) | Combines duplicate transactions into one clean entry using AI |
| [offline-sync.md](offline-sync.md) | Keeps the app usable without internet and syncs changes when you reconnect |
| [profile-settings.md](profile-settings.md) | Set your city, lifestyle tags, and income to personalise AI analysis |
| [push-notifications.md](push-notifications.md) | Sends a device notification when a receipt has finished being processed |
| [receipt-camera.md](receipt-camera.md) | Photographs or uploads a receipt and reads every line item automatically |
| [recurring-transactions.md](recurring-transactions.md) | Reminds you to log regular expenses and lets you record them with one tap |
| [refresh-app.md](refresh-app.md) | Clears the app cache and downloads the latest version |
| [scheduled-tasks.md](scheduled-tasks.md) | Runs email import, duplicate detection, and analysis retries automatically every day |
| [sheet-settings.md](sheet-settings.md) | View and manually sync your connected Google Sheet |
| [shortcut-automation.md](shortcut-automation.md) | Sends bank SMS messages to FundsFlee automatically via an iPhone Shortcut |
| [sms-text-parse.md](sms-text-parse.md) | Paste a bank SMS or payment alert and let AI create the transaction for you |
| [transaction-detail.md](transaction-detail.md) | View, edit, or delete a transaction and retry failed AI processing |
